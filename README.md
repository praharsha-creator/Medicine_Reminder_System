# Medicine Reminder System

## Project Overview

The Medicine Reminder System is a Python-based application designed to help users manage their medicine schedules efficiently. The system allows users to add, view, search, update, and delete medicine reminders while storing data permanently using SQLite database.

This project is developed using Object-Oriented Programming (OOP) concepts and demonstrates database management, file handling, and menu-driven application development in Python.

---

## Features

* Add new medicine reminders
* View all stored medicines
* Search medicine by name
* Update medicine reminder details
* Delete medicine records
* Export medicine data to CSV file
* Count total medicines stored
* SQLite database integration
* Object-Oriented Programming implementation
